# FIRSTFIGMAPROJECT
An interactive, single-page planetary experience built from a Figma design. Features smooth 2D rotational animations and click-based transitions between planets, recreated using HTML, CSS, and JavaScript and deployed on Vercel.https://www.figma.com/design/gqxBo6HhbB79b6jqon3C3H/karan-s-planets?node-id=0-1&amp;m=dev&amp;t=pQXss4Kx1GMWp6dp-1.
